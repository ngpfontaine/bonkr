Flexbox ¶

Examples

flex, row, shrink 0, wrap, spacing 10
.d-flx .flx-row .flx-wr-w .flx-children-s-0 .flx-children-spc-10
flex, row, nowrap, shrink 1, spacing 5
.d-flx .flx-row .flx-wr-n .flx-children-s-1 .flx-children-spc-5

Parent Container Classes

Define parent container as display flex, then add classes to customize or define children
  • .d-flx { display: flex; }
  • .flx-row { flex-direction: row; }
  • .flx-col { flex-direction: column; }
  • ‌
  • .flx-wr-n { flex-wrap: nowrap; }
  • .flx-wr-w { flex-wrap: wrap; }
  • .flx-wr-r > * { flex-wrap: wrap-reverse;}
  • ‌
  • .flx-spc-5 { margin-left: -5px; margin-right: -5px; }
  • .flx-spc-10 { margin-left: -10px; margin-right: -10px; }
  • .flx-spc-15 { margin-left: -15px; margin-right: -15px; }
  • .flx-spc-20 { margin-left: -20px; margin-right: -20px; }
  • ‌
  • .flx-children-0 > * { flex: 0; }
  • .flx-children-1 > * { flex: 1; }
  • ‌
  • .flx-children-g-0 > * { flex-grow: 0; }
  • .flx-children-g-1 > * { flex-grow: 1; }
  • .flx-children-s-0 > * { flex-shrink: 0; }
  • .flx-children-s-1 > * { flex-shrink: 1; }
  • ‌
  • .flx-children-spc-5 > * { margin: 0 5px 10px; }
  • .flx-children-spc-10 > * { margin: 0 10px 20px; }
  • .flx-children-spc-15 > * { margin: 0 15px 30px; }
  • .flx-children-spc-20 > * { margin: 0 20px 40px; }

Children Classes

Define sizing proportions of flex children
  • .flx-child-0 { flex: 0; }
  • .flx-child-0-5 { flex: 0.5; }
  • .flx-child-1 { flex: 1; }
  • .flx-child-1-5 { flex: 1.5; }
  • .flx-child-2 { flex: 2; }
  • .flx-child-2-5 { flex: 2.5; }
  • .flx-child-3 { flex: 3; }
  • .flx-child-3-5 { flex: 3.5; }
  • .flx-child-4 { flex: 4; }
